# Healthcare Patient Analysis Dashboard

Second Excel project: Analysis of 55,500+ patient records.

## Key Features
- KPIs: Total patients, avg age, stay, billing
- Charts: Trends, top conditions, top hospitals, admission types, demographics
- Slicers for condition, gender, year, etc.

Insights: Diabetes leads conditions, top hospitals by volume, avg stay 16 days

Tools: Excel PivotTables, Charts, Slicers
